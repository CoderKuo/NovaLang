### 解构声明

#### 位置解构（componentN）

```nova
// 列表解构
val (a, b, c) = [1, 2, 3]

// 跳过元素
val (first, _, third) = [1, 2, 3]

// 对象解构（需要 componentN 方法）
@data class Point(val x: Int, val y: Int)
val (x, y) = Point(3, 4)

// for 循环中解构
val pairs = [[1, "a"], [2, "b"]]
for ((num, str) in pairs) {
    println("$num -> $str")
}
```

#### 基于名称的解构声明

通过属性名而非位置进行解构，语法为 `val (localVar = propertyName, ...) = expr`。
核心优势：**顺序无关**，可以按任意顺序访问属性。

```nova
@data class User(val username: String, val email: String)
val user = User("alice", "alice@example.com")

// 名称解构：按属性名绑定，顺序任意
val (mail = email, name = username) = user
println(name)  // alice
println(mail)  // alice@example.com

// 反转声明顺序也能正确绑定
val (name2 = username, mail2 = email) = user  // 与声明顺序无关
```

**混合模式**：同一解构中可以混用名称解构和位置解构。

```nova
// 第一个按名称，第二个按位置（component2）
val (mail = email, second) = user

// 第一个按位置（component1），第二个按名称
val (first, mail = email) = user
```

**带跳过**：

```nova
val (_, name = username) = user    // 跳过第一个，按名称取 username
```

**for 循环中使用**：

```nova
val users = listOf(User("alice", "a@e.com"), User("bob", "b@e.com"))
for ((name = username, mail = email) in users) {
    println("$name: $mail")
}

// 不同顺序
@data class Item(val name: String, val price: Int)
val items = listOf(Item("apple", 10), Item("banana", 20))
var total = 0
for ((cost = price, label = name) in items) {
    total = total + cost
}
```

**非 data class 也支持**（通过字段访问）：

```nova
class Config(val host: String, val port: Int)
val cfg = Config("localhost", 8080)
val (h = host, p = port) = cfg
```

## 函数

### 基本函数

```nova
fun greet(name: String): String {
    return "Hello, $name!"
}
```

### 表达式体函数

```nova
fun double(x: Int) = x * 2
fun max(a: Int, b: Int) = if (a > b) a else b
```

### 函数声明后立即调用（IIFE）

函数声明后紧跟 `()` 可立即调用，函数仍注册到作用域：

```nova
fun answer(): Int { return 42 }()   // 声明 answer 并立即调用，返回 42
answer()                             // 之后仍可再次调用
```

### 默认参数

```nova
fun greet(name: String, greeting: String = "Hello") {
    println("$greeting, $name!")
}

greet("Alice")            // Hello, Alice!
greet("Bob", "Hi")        // Hi, Bob!
```

### 可变参数（vararg）

```nova
fun sum(vararg numbers: Int): Int {
    var total = 0
    for (n in numbers) {
        total = total + n
    }
    return total
}

sum(1, 2, 3)    // 6
```

### 泛型函数

```nova
fun <T> identity(value: T): T = value
fun <T> listOfSingle(item: T): List = listOf(item)
```

### 扩展函数

为已有类型添加方法，无需修改原类型：

```nova
fun String.exclaim() = this + "!"
fun Int.isEven() = this % 2 == 0

"Hello".exclaim()    // "Hello!"
4.isEven()           // true
```

### 命名参数

调用时指定参数名，可以改变参数顺序：

```nova
fun createUser(name: String, age: Int, active: Boolean = true) = "$name($age)"

createUser("Alice", 25)                   // 位置参数
createUser(age = 30, name = "Bob")        // 命名参数（可乱序）
createUser("Charlie", active = false, age = 20)  // 混合使用
```

### Infix 函数

用 `infix` 修饰的扩展函数支持中缀调用语法：

```nova
infix fun Int.power(exp: Int): Int {
    var result = 1
    for (i in 1..exp) { result = result * this }
    return result
}

val r = 2 power 10     // 等价于 2.power(10)，结果 1024
val r2 = 2.power(10)   // 点调用也可以
```

### function 关键字

`function` 是 `fun` 的别名，兼容 JavaScript 风格：

```nova
function greet(name) {
    return "Hello, $name!"
}
```

### Spread 操作符

在列表字面量和函数调用中展开集合：

```nova
val a = [1, 2]
val b = [0, ..a, 3]     // [0, 1, 2, 3]  — .. 语法
val c = [0, *a, 3]      // [0, 1, 2, 3]  — * 语法（等价）

// 函数调用中的 Spread
fun sum(a: Int, b: Int, c: Int) = a + b + c
val args = [1, 2, 3]
sum(..args)              // 6 — 展开列表作为位置参数
```

```nova
val String.wordCount = this.split(" ").size()

"Hello Nova World".wordCount  // 3
```

```nova
    return result
```

## 类与对象

### 类定义

```nova
class Person(val name: String, var age: Int) {
    fun greet() = "Hi, I'm $name, $age years old"

    fun toString() = "Person($name, $age)"
}

val p = Person("Alice", 30)
println(p.greet())      // Hi, I'm Alice, 30 years old
println(p.name)         // Alice
p.age = 31              // var 属性可修改
```

### 初始化块

`init { ... }` 块在实例创建时执行，可访问主构造器参数和前面已声明的属性。一个类中允许多个 init 块，它们与属性初始化器按声明顺序交织执行。

```nova
class Validator(val value: Int) {
    init {
        require(value >= 0, "Value must be non-negative")
    }
}
```

多个 init 块与属性初始化器按声明顺序执行：

```nova
class Foo(val x: Int) {
    val a = x + 1
    init { println("a = $a") }       // 可访问 x 和 a
    val b = a * 2
    init { println("b = $b") }       // 可访问 x, a, b
}

val f = Foo(3)
// 输出:
// a = 4
// b = 8
```

init 块中可使用控制流和调用方法：

```nova
class Config(val port: Int) {
    var status = ""
    init {
        if (port > 0) {
            status = "valid"
        } else {
            status = "invalid"
        }
    }
}
```

无主构造器参数的类也可使用 init 块：

```nova
class Counter {
    var count = 0
    init {
        count = 100
    }
}
```

次级构造器委托到主构造器时，init 块由主构造器触发：

```nova
class Point(val x: Int, val y: Int) {
    init { println("Point($x, $y)") }
    constructor(v: Int) : this(v, v)
}

val p = Point(5)   // 输出: Point(5, 5)
```

### 次级构造器

```nova
class Rectangle(val width: Int, val height: Int) {
    constructor(size: Int) : this(size, size)  // 委托到主构造器
}

val square = Rectangle(5)     // 使用次级构造器
val rect = Rectangle(3, 4)    // 使用主构造器
```

### Data Class（@data 注解）

自动生成 `toString`、`equals`、`hashCode`、`copy`、`componentN` 方法：

```nova
@data class User(val name: String, val age: Int)

val user = User("Alice", 30)
println(user)                          // User(name=Alice, age=30)
val older = user.copy(age = 31)        // User(name=Alice, age=31)
val (name, age) = user                 // 解构
```

### Data Object（@data object）

`data object` 是无字段的单例对象，自动生成一致的 `toString`（返回纯类名）、`equals`、`hashCode`。
常与 `sealed interface` 配合使用，表示无数据的状态分支：

```nova
sealed interface Result
@data class Success(val data: String) : Result
@data object Error : Result

println(Success("ok"))   // Success(data=ok)
println(Error)            // Error（纯类名，无括号）

// when 分支匹配
fun describe(r: Result) = when(r) {
    is Success -> "成功: " + r.data
    is Error -> "失败"
    else -> "未知"
}
```

`data object` 也可以带方法和属性：

```nova
@data object AppConfig {
    val version = "1.0"
    fun info() = "App v" + version
}

println(AppConfig)        // AppConfig
println(AppConfig.info()) // App v1.0
```

## 控制流

### if-else

作为语句或表达式使用：

```nova
// 语句
if (x > 0) {
    println("positive")
} else if (x < 0) {
    println("negative")
} else {
    println("zero")
}

// 表达式
val result = if (x > 0) "positive" else "negative"
```

### when 表达式

```nova
// 值匹配
when (x) {
    1 -> "one"
    2 -> "two"
    3, 4 -> "three or four"
    in 5..10 -> "between 5 and 10"
    else -> "other"
}

// 类型匹配
when (obj) {
    is String -> "String of length ${obj.length()}"
    is Int -> "Int: $obj"
    is List -> "List of size ${obj.size()}"
    else -> "Unknown"
}

// 无参 when（替代 if-else if 链）
when {
    x < 0 -> "negative"
    x == 0 -> "zero"
    else -> "positive"
}
```

### when Guard Conditions

在 when 分支条件后添加 `if guardExpr`，只有条件匹配**且** guard 为真时才进入该分支，否则跳到下一分支：

```nova
when (animal) {
    is Cat if !animal.isHungry -> "feed cat"   // Cat 且不饿
    is Cat -> "hungry cat"                      // Cat 但饿了（guard 失败后落到这里）
    is Dog -> "walk dog"
    else -> "unknown"
}

// 带范围 + guard
when (score) {
    in 90..100 if honors -> "A+"
    in 90..100 -> "A"
    in 80..89 -> "B"
    else -> "C"
}

// 无参 when + guard
fun classify(n: Int) = when {
    n > 0 if isEven(n) -> "even positive"
    n > 0 -> "odd positive"
    else -> "non-positive"
}

// 多条件 + guard
when (x) {
    1, 2 if flag -> "1 or 2 with flag"
    1, 2 -> "1 or 2"
    else -> "other"
}
```

guard 表达式支持任意表达式：比较、方法调用、逻辑运算（`&&`/`||`）、函数调用、`in` 范围检查、`is` 类型检查等。

### Non-local break / continue

在 `forEach` 等高阶函数的 lambda 内可以使用 `break` 和 `continue`，效果与普通 for 循环一致：

```nova
// continue — 跳过当前元素，继续下一次迭代
[1, 2, 0, 3].forEach {
    if (it == 0) continue
    println(it)   // 输出 1, 2, 3（跳过 0）
}

// break — 终止整个遍历
[1, 2, 3, 4, 5].forEach {
    if (it == 3) break
    println(it)   // 输出 1, 2（在 3 处停止）
}

// forEachIndexed 同样支持
["a", "b", "c", "d"].forEachIndexed { i, v ->
    if (i == 2) break
    println("$i: $v")   // 输出 0: a, 1: b
}

// 嵌套 forEach — break 只影响最内层
[1, 2].forEach { outer ->
    ["a", "b", "c"].forEach { inner ->
        if (inner == "b") break   // 只终止内层 forEach
        println("$outer$inner")   // 输出 1a, 2a
    }
}
```

`return@forEach` 也可用作 `continue` 的替代语法（跳过当前 lambda 调用）。

支持的高阶函数：`forEach`、`forEachIndexed`、Map 的 `forEach`。

### for 循环

```nova
// 遍历范围

    println(i)

// 半开区间

// 遍历字符串（逐字符迭代）

}

// 解构遍历
for ((key, value) in #{"a": 1, "b": 2}) {
    println("$key = $value")
}
```

### try-catch-finally


### break 和 continue

```nova

    if (i == 5) break       // 跳出循环
    if (i % 2 == 0) continue // 跳过偶数
    println(i)
}
```

### 标签化的 break/continue/return

```nova
outer@ for (i in 1..5) {
    for (j in 1..5) {
        if (i * j > 10) break@outer  // 跳出外层循环
    }
}

fun processItems(items: List) {
    items.forEach lit@ { item ->
        if (item == null) return@lit  // 跳过当前 Lambda 迭代
        println(item)
    }
}
```

### guard 语句

安全解包，null 时执行 else 分支（必须包含 return/throw/break/continue）：

```nova
fun process(value: Int?): Int {
    guard val x = value else { return -1 }
    return x * 2
}

process(5)      // 10
process(null)   // -1
```

### use 语句

类似 try-with-resources，自动关闭资源：

```nova
use (val reader = openFile("data.txt")) {
    val content = reader.readLines()
    println(content)
}
// reader 在此自动关闭

// 多资源
use (val conn = getConnection(), val stmt = conn.createStatement()) {
    stmt.execute("SELECT * FROM users")
}
```

### 运算符

#### 算术运算符

| 运算符 | 说明 |
|--------|------|

| `-x` | 一元负号 |

| `--x` `x--` | 自减 |

#### 比较运算符

| 运算符 | 说明 |
|--------|------|
| `==` `!=` | 结构相等/不等 |
| `===` `!==` | 引用相等/不等 |
| `<` `>` `<=` `>=` | 大小比较 |

#### 逻辑运算符

| 运算符 | 说明 |
|--------|------|
| `&&` | 逻辑与（短路求值） |
| `\|\|` | 逻辑或（短路求值） |
| `!` | 逻辑非 |

#### 位运算符

| 运算符 | 说明 |
|--------|------|
| `&` | 按位与 |
| `\|` | 按位或 |
| `^` | 按位异或 |
| `~` | 按位取反（一元） |
| `<<` | 左移 |
| `>>` | 算术右移（带符号） |
| `>>>` | 逻辑右移（无符号） |

位运算符仅支持 `Int` 和 `Long` 类型。

```nova
val a = 0xFF00
val b = 0x0FF0

a & b               // 0x0F00 (按位与)
a | b               // 0xFFF0 (按位或)
a ^ b               // 0xF0F0 (按位异或)
~0                   // -1     (按位取反)

1 << 10              // 1024   (左移)
1024 >> 3            // 128    (算术右移)
(-1) >>> 16          // 65535  (逻辑右移，高位补 0)
```

运算符优先级（从低到高）：`||` → `&&` → `|` → `^` → `&` → `==`/`!=` → `<`/`>` → `<<`/`>>`/`>>>` → `+`/`-` → `*`/`/`/`%` → `~`/`!`/`-`（前缀）

```nova
// 位运算常见用法
val color = 0xFF8040
val r = (color >> 16) & 0xFF    // 255（提取红色分量）
val g = (color >> 8) & 0xFF     // 128（提取绿色分量）
val b = color & 0xFF            // 64 （提取蓝色分量）

val flags = 0b1010
flags & 1                       // 0（检测奇偶）
16 & 15                         // 0（判断 2 的幂：n & (n-1) == 0）
(-1) & 0xFF                     // 255（取低 8 位）
```

#### 包含运算符

`in` 和 `!in` 可用于范围检查和集合包含检查：

```nova
val x = 5
x in 1..10           // true（范围包含检查）
x !in 20..30         // true（范围不包含）
3 in [1, 2, 3, 4]   // true（列表包含）

// 在 if 和 when 中使用
if (score in 90..100) "A" else "B"

val period = when {
    hour in 0..<6 -> "深夜"
    hour in 6..<12 -> "上午"
    hour in 12..<18 -> "下午"
    else -> "晚间"
}
```

#### 赋值运算符

| 运算符 | 说明 |
|--------|------|
| `=` | 赋值 |
| `+=` `-=` `*=` `/=` `%=` | 算术复合赋值 |
| `&=` `\|=` `^=` `<<=` `>>=` `>>>=` | 位运算复合赋值 |
| `??=` | 空值合并赋值（仅当变量为 null 时赋值） |
| `\|\|=` | 逻辑或赋值（仅当变量为 falsy 时赋值） |
| `&&=` | 逻辑与赋值（仅当变量为 truthy 时赋值） |

```nova
var x: Int? = null
x ??= 42          // x = 42（因为 x 是 null）
x ??= 100         // x = 42（不变，因为 x 已非 null）
```

### 空安全

| 运算符 | 说明 |
|--------|------|
| `?.` | 安全调用（null 时返回 null） |
| `?[]` | 安全索引（null 时返回 null） |
| `?:` | Elvis 运算符（null 时使用默认值） |
| `!!` | 非空断言（null 时抛出异常） |

```nova
val name: String? = null
name?.length()         // null（不会抛异常）
name ?: "Unknown"      // "Unknown"
// name!!              // 抛出异常

val list: List? = null
list?[0]               // null
```

### 类型操作

| 运算符 | 说明 |
|--------|------|
| `is` | 类型检查 |
| `!is` | 类型否定检查 |
| `as` | 类型转换（失败抛异常） |
| `as?` | 安全类型转换（失败返回 null） |

```nova
if (obj is String) {
    println(obj.length())  // 智能转换
}

val s = obj as? String     // 安全转换，失败返回 null
val n = null as String?    // 可空类型转换，null 允许通过
val x = (null as String?)?.toUpperCase() ?: "UNKNOWN"  // 配合安全调用和 elvis
// 注意：null as String（非可空）会抛异常
```

#### as 数值类型转换

`as` 支持数值类型之间的转换（截断/扩展/转字符串）：

```nova
3.14 as Int      // 3（截断小数）
42 as Double     // 42.0
42 as Long       // 42L
42L as Int       // 42
42 as String     // "42"
"123" as Int     // 123（字符串解析）
true as String   // "true"
```

> `as?` 不做数值转换，只做 instanceof 检查。`42 as? String` 返回 null。

#### as 优先级

`as` 的优先级**高于四则运算**（与 Kotlin 一致），可直接在表达式中使用：

```nova
// as 先于 * 和 + 绑定
val result = base + (value ?: 0) as Int * 28
// 等价于: base + (((value ?: 0) as Int) * 28)

// 索引内也支持 as
list[x as Int]
```

当用户定义的类名与内置类型冲突时，可用 `nova.` 前缀访问内置类型：

```nova
sealed class Result                       // 用户定义的 Result
class Success(val data: String) : Result()

val userR: Result = Success("ok")
val builtinR = Ok(42)                     // 内置 Ok() 创建内置 Result

userR is Result        // true（用户定义的 Result）
builtinR is nova.Result // true（内置 Result）
42 is nova.Int          // true（内置 Int 类型）
```

## 运算符重载

通过定义约定方法名来重载运算符：

| 运算符 | 方法名 |
|--------|--------|
| `a + b` | `plus(other)` |
| `a - b` | `minus(other)` |
| `a * b` | `times(other)` |
| `a / b` | `div(other)` |
| `a % b` | `rem(other)` |
| `-a` | `unaryMinus()` |
| `+a` | `unaryPlus()` |
| `a++` | `inc()` |
| `a--` | `dec()` |
| `a[i]` | `get(index)` |
| `a[i] = v` | `set(index, value)` |
| `a < b` 等 | `compareTo(other)` |

```nova
class Vec2(val x: Double, val y: Double) {
    fun plus(other: Vec2) = Vec2(x + other.x, y + other.y)
    fun minus(other: Vec2) = Vec2(x - other.x, y - other.y)
    fun times(scalar: Double) = Vec2(x * scalar, y * scalar)
    fun unaryMinus() = Vec2(-x, -y)
    fun toString() = "($x, $y)"
}

val a = Vec2(1.0, 2.0)
val b = Vec2(3.0, 4.0)
println(a + b)       // (4.0, 6.0)
println(a * 2.0)     // (2.0, 4.0)
println(-a)          // (-1.0, -2.0)
```

`compareTo` 返回 Int（<0, 0, >0），自动支持 `<`、`>`、`<=`、`>=`：

```nova
class Version(val major: Int, val minor: Int) {
    fun compareTo(other: Version): Int {
        val diff = major - other.major
        if (diff != 0) return diff
        return minor - other.minor
    }
}

Version(2, 0) > Version(1, 9)   // true
```

## 异步编程

NovaLang 提供了一套完整的异步并发原语，基于 JVM 线程池实现。

### async / await

`async { }` 启动异步计算，返回 `Future` 对象；`await` 关键字阻塞等待 Future 的结果：

```nova
val future = async {
    // 在后台线程执行
    heavyComputation()
}

// 做其他工作...
val result = await future   // 阻塞等待结果
println(result)
```

Future 对象还提供以下成员方法：

| 方法 | 说明 |
|------|------|
| `future.get()` | 等同于 `await future`，阻塞等待结果 |
| `future.cancel()` | 取消异步计算 |
| `future.isDone` | 是否已完成 |
| `future.isCancelled` | 是否已取消 |

### launch

`launch { }` 启动一个"发射即忘"的异步任务，返回 `Job` 对象（不携带返回值）：

```nova
val job = launch {
    println("后台任务执行中")
}

job.join()       // 等待任务完成
job.cancel()     // 取消任务
job.isActive     // 是否仍在运行
job.isCompleted  // 是否已完成
```

### parallel

`parallel` 并行执行多个 lambda，等待全部完成后返回结果列表：

```nova
val results = parallel(
    { fetchUser() },
    { fetchOrders() },
    { fetchConfig() }
)
// results 是三个任务返回值的列表
```

### withTimeout

`withTimeout(millis) { }` 带超时执行，超时后抛出异常：

```nova
val data = withTimeout(3000) {
    slowNetworkRequest()
}
```

### awaitAll / awaitFirst

```nova
val f1 = async { task1() }
val f2 = async { task2() }
val f3 = async { task3() }

// 等待所有 Future 完成，返回结果数组
val all = awaitAll([f1, f2, f3])

// 返回第一个完成的 Future 的结果
val first = awaitFirst([f1, f2, f3])
```

### delay

`delay(millis)` 在异步上下文中暂停当前线程（不可在主线程调用）：

```nova
launch {
    delay(1000)
    println("1 秒后执行")
}
```

### Channel

`Channel(capacity?)` 创建线程安全的并发通道，用于协程间通信：

```nova
val ch = Channel()         // 无界通道
val ch2 = Channel(10)      // 容量为 10 的有界通道

// 生产者
launch {
    for (i in 1..5) {
        ch.send(i)
    }
    ch.close()
}

// 消费者
launch {
    for (item in ch) {
        println(item)
    }
}
```

Channel 成员方法：

| 方法 | 说明 |
|------|------|
| `send(value)` | 发送数据（通道已满时阻塞） |
| `receive()` | 接收数据（通道为空时阻塞） |
| `tryReceive()` | 非阻塞接收，无数据返回 null |
| `receiveTimeout(ms)` | 带超时的接收 |
| `close()` | 关闭通道 |
| `isEmpty` / `size` / `isClosed` | 状态查询 |

### Mutex

`Mutex()` 创建互斥锁，保护共享资源：

```nova
val mutex = Mutex()
var count = 0

// 手动加锁/解锁
mutex.lock()
count = count + 1
mutex.unlock()

// 推荐：使用 withLock 自动管理锁
mutex.withLock {
    count = count + 1
}

mutex.tryLock()   // 尝试获取锁（非阻塞）
mutex.isLocked    // 是否已被锁定
```

### 原子类型

线程安全的原子变量，无需加锁：

```nova
val counter = AtomicInt(0)
counter.incrementAndGet()       // 1
counter.addAndGet(5)            // 6
counter.get()                   // 6
counter.compareAndSet(6, 10)    // true

val ref = AtomicRef("hello")
ref.set("world")
ref.get()                       // "world"
```

### 异步错误处理

`async` 中抛出的异常会在 `await` 时传播到调用方：

```nova
val future = async {
    throw Exception("出错了")
}

try {
    await future
} catch (e) {
    println(e)   // "出错了"
}
```

`launch` 中未捕获的异常会输出到日志。可用 `runCatching` 结合异步使用：

```nova
val future = async {
    runCatching {
        riskyOperation()
    }
}

val result = await future
when (result) {
    is Ok -> println(result.value)
    is Err -> println("失败: ${result.error}")
}
```
